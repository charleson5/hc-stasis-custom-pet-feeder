# hc-stasis-custom-pet-feeder
This is my repository for my Pet Feeder custom project for HackClub Stasis

Log 1: 4/2/26.
Planned out all the electronics for the pet feeder in a program called CirkitBuilderIDE.
Wrote the code for the arduino uno chip, and simulated it to confirm it works.
Uploaded Image of Electronic layout to the github repository. 
(see circuit_image.png)
List of Electronics: Arduino Uno R3, Breadboard, 2x16 16pin LCD Display, Bipolar Stepper Motor (NEMA 17), A4988 Stepper Motor Driver Carrier, DS1307 RTC, Trimmer Potentiometer, Pushbutton, External 12V DC Power Supply.
